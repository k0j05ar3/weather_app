# Weather App 🌤️

A simple weather application built with **HTML, CSS, and JavaScript** that fetches real-time weather data using the **WeatherAPI**.

## Features

* 🔍 Search weather by city
* 🌡️ Display current temperature
* ☁️ Show current weather conditions
* 📍 Display location information
* ⚡ Fetch live weather data using the Fetch API
* 🚨 Basic error handling for invalid searches

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* Async/Await
* Fetch API
* WeatherAPI

## What I Learned

This project helped me practice:

* Making HTTP requests with the Fetch API
* Working with asynchronous JavaScript (`async`/`await`)
* Parsing JSON data
* Manipulating the DOM
* Handling API errors
* Organizing JavaScript into reusable functions

## Getting Started

1. Clone the repository.
2. Sign up for a free API key from WeatherAPI.
3. Replace the placeholder API key in the JavaScript file with your own.
4. Open `index.html` in your browser.
